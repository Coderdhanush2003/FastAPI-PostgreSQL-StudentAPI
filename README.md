# Student API

A REST API built with FastAPI, PostgreSQL, and SQLAlchemy ORM.

## Features
- Add students
- Get all students
- Get student by ID
- Update student
- Delete student

## Tech Stack
- FastAPI
- PostgreSQL
- SQLAlchemy ORM
- Pydantic
  
## How to Run

1. Clone the repository
   git clone https://github.com/Coderdhanush2003/FastAPI-PostgreSQL-StudentAPI.git

2. Install dependencies
   pip install fastapi sqlalchemy pg8000 pydantic

3. Set up PostgreSQL and create a database

4. Update the database URL in database.py

5. Run the server
   fastapi dev main.py

6. Open API docs
   http://127.0.0.1:8000/docs
